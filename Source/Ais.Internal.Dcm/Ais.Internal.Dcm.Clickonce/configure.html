AIS Windows Azure Media Services Management Tool

  1. Install the AIS Windows Azure Media Services Management application by clicking here

    2. Note: Windows 7 Users can skip step 2 & 3- they are exclusively for windows 8/8.1 users

  2. click on the More Info text
  3. click on "Run Anyway"

    4. Note :windows 8/8.1 Users can skip step #4 - this is exclusive for windows 7 users

  4. Executing the setup can give you a security alert. Please let the setup run.
  5. click on install button

    6. Note:Windows 7 users can skip step 6 & 7 - they are exclusive for windows 8/8.1 Users

  6. click on the More Info text
  7. click on "Run Anyway"
  8. Application will be downloaded.
  9. This setup will install "Windows Azure Media Services Management Tool" application in your system
  10. Once the application starts click on the Account Setting button verify if the URL is like :

    http://yoursite.azurewebsites.net
